## Fonctionnalités principales

- Création de compte et connexion utilisateur
- Consultation des événements sportifs disponibles
- Création d’un événement sportif
- Rejoindre / quitter un événement
- Navigation fluide entre les écrans
- Consommation d’une API REST sécurisée

---

## Stack technique

- **Langage** : JavaScript / TypeScript  
- **Framework** : React Native  
- **Outil** : Expo  
- **Communication API** : HTTP / REST  
- **Outils** : Git, GitHub  

---

## Installation et lancement du projet

1. Cloner le dépôt
git clone https://github.com/aminebacha35/tempo-front.git
cd tempo-front
Installer les dépendances

npm install
ou

yarn install
Lancer l’application

npx expo start
L’application peut être lancée sur :

un émulateur

un appareil physique via l’application Expo Go
